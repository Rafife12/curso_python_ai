# 💡 Cursor com Python: Desenvolvimento Inteligente com IA

Este repositório contém exemplos práticos e anotações baseadas no curso **"Cursor com Python: Desenvolvimento Inteligente com IA"** oferecido pela Santander Open Academy.

## 📌 Sobre o Curso

Curso gratuito que ensina como usar o **Cursor**, editor de código com Inteligência Artificial, para desenvolvimento em Python, com foco em produtividade e boas práticas.

## 📁 Conteúdo do Repositório

- `main.py`: Exemplo de uso do Cursor para geração, refatoração e consulta de código em Python.
- `requirements.txt`: Dependências utilizadas nos exemplos (formatadores, linters, etc).
- Anotações sobre prompts, refatorações e comandos úteis com IA.

## 🧠 Exemplos de Prompts para o Cursor

```txt
# Gere uma função para ordenar uma lista de números
# Refatore essa função usando list comprehension
# Adicione docstring explicando a função abaixo
# Teste essa função com três entradas diferentes
```

## 🛠 Tecnologias e Ferramentas

- Python 3.11+
- Cursor IDE (https://www.cursor.com)
- Git + GitHub
